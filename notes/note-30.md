## Tools to establish that a program "works"

* Precondition: something that must be true before the program starts
* Postcondition: someting that will be true when the program completes, assuming
  the precondition was true
* Invariant: something that remains true at the beginning and at the end of each
  iteration of a loop
* Ranking function: something that strictly decreases in each iteration of a
  loop
